## Setup

```bash
$ git clone https://github.com/2gbeh/marina-cms.git

$ cd marina-cms

$ npm install

$ composer install

# OPTIONAL copy .env credentials

$ npm run dev

$ php artisan serve
```

> Local: http://localhost:8000
